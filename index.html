Password Generator Bookmarklet

I needed something like Clipperz password generator, just more accessible. I had to write it myself because the only thing I found were password generator which worked as a function of the current URL and a master password and are therefore bad because for all I know the hoster can see my password and maybe even deduct the algorithm.

So this generates totally random passwords with configurable charsets. There you go :)
